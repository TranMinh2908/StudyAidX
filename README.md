## 🌟 Features

<div align="center">
  <img src="https://i.imgur.com/your-features-animation.gif" alt="StudyAidX Features" width="600">
</div>

- 🔍 **Quiz Data Extraction**: Easily extract and analyze quiz content.
- 🤖 **AI-Powered Answering**: Get intelligent answer suggestions using advanced AI.
- ⏱️ **Customizable Timer**: Set study sessions with a sleek countdown timer.
- 🎨 **Theme Customization**: Personalize your study environment with custom themes.
- 🎵 **Study Music Player**: Built-in music player to boost your concentration.
- 📊 **Answer Analysis**: Gain insights with frequency analysis of your answers.
- 🔄 **Auto-Submit Option**: Save time with automatic quiz submission.
- 💾 **Answer Management**: Save and load your answers for future reference.
- 🖍️ **Answer Highlighting**: Visually emphasize important information.
- 🚜 **Farm Mode**: Efficient practice mode for repetitive learning.

## 🚀 Getting Started

<div align="center">
  <img src="https://i.imgur.com/your-installation-animation.gif" alt="Installation Process" width="400">
</div>

1. Install the Tampermonkey browser extension.
2. Create a new script in Tampermonkey.
3. Copy the entire StudyAidX script and paste it into the new Tampermonkey script.
4. Save the script and enable it.
5. Navigate to your learning management system (LMS) and enjoy StudyAidX!

## 🎮 How to Use

<div align="center">
  <img src="https://i.ibb.co/9cT1f6x/Screenshot-2024-10-25-201217.png" alt="StudyAidX Usage" width="500">
</div>

1. **Activate StudyAidX**: Press `Ctrl + Q` to toggle the StudyAidX menu.
2. **Extract Quiz Data**: Click the "Extract" button to gather quiz information.
3. **Answer Selection**: Use AI or input your own answers for automatic selection.
4. **Customize Experience**: Adjust themes, set timers, and personalize your study session.
5. **Analyze & Improve**: Utilize answer analysis and highlighting features to enhance your learning.
